Disparate data in, unified data out

InterMine integrates biological data sources, making it easy to query and analyse data.

InterMine is open source (LGPL 2.1) and free to use.

Around thirty different InterMines exist worldwide, covering a broad range of organisms.

C. cajan (CowpeaMine) Archae (IndigoMine) H. sapiens (TargetMine) M. truncatula (LegumeMine) Viridiplantae (PhytoMine) D. melanogaster (Shaare) B. taurus (BovineMine) H. sapiens (HumanMine) P. tenuis (PlanMine) M. truncatula f. tricycla (MedicMine) D. melanogaster (FlyMine) R. norvegicus (RatMine) A. thaliana (ThaleMine) C. arietinum (SoyMine) M. musculus (MouseMine)

There's an InterMine for (almost) anyone

With InterMine you can search across data for one or many organisms, and jump between different organisms based on homology.

This is a sample of the organisms you can find in each mine - click on one to go straight to that InterMine.

InterMine Features

Dynamic data tables

InterMine's tables allow you to easily drill down into data. It's easy to filter data, add additional columns, navigate to report pages.

Try it yourself!

Try clicking on the icon in the ontology term namespace and filtering the column to show only one term.

Run queries in many languages

Try clicking on the "Generate Python code" button - or use the dropdown to choose a different language. You can then easily copy/paste the queries to your scripts.
