InterMine Features

Dynamic Data Tables

InterMine's tables allow you to easily drill down into data. It's easy to filter data, add additional columns, navigate to report pages.

Try it yourself!

Try clicking on the  icon in the ontology term namespace and filtering the column to show only one term.

Run queries in many languages

Try clicking on the "Generate Python code" button - or use the dropdown to choose a different language. You can then easily copy/paste the queries to your scripts. Shown in table: GO terms related to the Drosophila gene "zen". Data drawn from FlyMine.
